Навыки

Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

SASS/SCSS

Препроцессор CSS. Используется данный язык для того, чтобы взаимодействовать со стилями. Цель применения: упростить и структурировать код, а также позволяет работать быстрее

Bootstrap

Фреймворк Bootstrap представляет собой набор CSS и JavaScript файлов, что позволяет верстать сайты в несколько раз быстрее, чем это можно выполнить на «чистом» CSS и JavaScript.

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

React

Эта библиотека позволяет создавать web-приложения, что позволяет создать максимально интерактивный продукт именно под ваши цели

Redux

Redux хранит состояние всего вашего приложения в одном месте, включает данные из вашего API, состояния навигации, информацию о пользователе, переключенное состояние кнопки и т.д.

Wordpress

WordPress — это самый известный и наиболее удобный для работы с контентом движок(CMS). Заказчик может самостоятельно управлять контентом на сайте через админпанель

Gulp

Gulp - это инструмент для автоматизации рутинных задач, которые возникают при веб-разработке, что позволяет сильно ускорись выполнение задания

Git

Git — это система управления версиями. У Git основная задача: хранить информацию о всех изменениях в вашем коде, начиная с самой первой строчки.
